# ml-prac

1) lab1_1.py - часть 1 задания
2) lab1_2.py - части 2 и 3 задания
3) lab01_pandas_knn_trees.ipynb - основные результаты вычислений.
   Сами программы в файл не копировал, т.к. вычисления не быстрые и файл будет
   долго открываться.

Для того, чтобы программы запустились, необходимо, чтобы в той же папке
находились файлы с данными:
2008.csv.bz2, airports.csv, amazon.csv
Так как загрузка по http в процессе выполнения программы ещё более увеличит
время её работы.
Поэтому после клонирования репозитория, необходимо скачать данные файлы по
адресам, указанным в lab01_pandas_knn_trees.ipynb в папку репозитория.

После этого можно осуществлять запуск программ:
python3 lab1_1.py ../../machinelearning_data/lab1_1sem/2008.csv.bz2 ../../machinelearning_data/lab1_1sem/airports.csv
python3 lab1_2.py ../../machinelearning_data/lab1_1sem/amazon.csv

И сверять результаты с отчётом.
